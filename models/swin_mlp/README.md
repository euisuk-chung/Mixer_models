## Swin MLP (Pytorch)

Swin transformer: Hierarchical vision transformer using shifted windows (<a href="https://arxiv.org/abs/2108.01072"> Link </a>)

<img src="./swin_mlp.png" width="800px"></img>

Swin MLP is model introduced in Swin Transformer paper. It is an Swin Tranformer like model, using MLP block instead of self-attention block.
